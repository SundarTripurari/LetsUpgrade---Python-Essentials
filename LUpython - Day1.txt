2. What are code environments and Types of code environments?

Code Environment is a place where you can write down the program, compile the code and see the output of code at same place.

Types of Code Environments:

1. Text Editors eg: Notepad, Visual Studio Code
2. Full IDEs eg: Spyder, Pycharm
3. Notebook Environments eg: Jupyter notebook, Google Colab
